## Getting Started

To get a local copy up and running follow these simple example steps.

Clone the project by using terminal:

```
git clone https://github.com/adamteddychang/.git
```

cd into the directory:

```
cd space-travelers
```

install node modules:

```
npm install
```

start live server:

```
npm start
```

for testing:

```
npm run test
```
